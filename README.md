# Bolt
[![Build Status](https://travis-ci.org/sakkayaphab/bolt.svg?branch=master)](https://travis-ci.org/sakkayaphab/bolt)


## Installation
```sh
git clone https://github.com/sakkayaphab/bolt.git
cmake .
make
```


## Usage
```sh
cd bin
./bolt
```
for [user guide][UserGuide]
