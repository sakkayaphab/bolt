# Bolt

[![GitHub release (latest by date)](https://img.shields.io/github/v/release/sakkayaphab/bolt)](https://github.com/sakkayaphab/bolt/releases)


## Installation
```sh
git clone https://github.com/sakkayaphab/bolt.git
cmake .
make
```


## Usage
```sh
cd bin
./bolt
```
